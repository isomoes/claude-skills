# Claude Code Skills

This directory contains custom skills for Claude Code agents to enhance their capabilities for academic paper writing and thesis management.

## What are Skills?

Skills are specialized capabilities that extend Claude Code's functionality. They are loaded and executed by the agent to perform specific tasks more effectively.

## Available Skills

The skills in this repository are designed to support:

- **Academic Paper Writing**: Tools for searching, analyzing, and citing academic papers
- **Weekly Reporting**: Automated generation of research progress reports
- **Paper Management**: Finding related papers and managing references
- **Commit Management**: Standardized commit messages with changelog updates

## Usage

Skills are automatically loaded by Claude Code when configured in the MCP (Model Context Protocol) settings. They can be invoked using the `/skill-name` command syntax.

## Structure

Each skill is defined with:
- A unique identifier
- Description of its purpose
- Required parameters and tools
- Implementation logic

## For Agents

These skills are designed to be invoked by Claude Code agents to streamline workflows for academic writing, research management, and version control in the context of thesis and paper preparation.
